# Projeto Final TCC - Dynacoop

## 📌 Descrição
Este repositório contém o **projeto final de TCC** desenvolvido para a Dynacoop, com foco em **Microsoft Dynamics 365**.  
O trabalho consistiu na criação de **plugins customizados** para estender funcionalidades do CRM, automatizar processos e atender requisitos específicos de negócio.

## 🛠️ Tecnologias Utilizadas
- Microsoft Dynamics 365
- .NET / C#
- SDK do Dynamics 365
- Visual Studio
- JavaScript (para customizações adicionais)

## 🚀 Funcionalidades
- Desenvolvimento de plugins customizados para eventos do Dynamics 365.
- Automação de regras de negócio específicas da Dynacoop.
- Manipulação de entidades e registros do CRM.
- Extensão de funcionalidades padrão do Dynamics.

## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido como **Trabalho de Conclusão de Curso (TCC)**, com o objetivo de aplicar conhecimentos adquiridos em:
- Desenvolvimento de soluções corporativas com Dynamics 365.
- Criação de plugins e integrações personalizadas.
- Boas práticas de programação em .NET e C#.

## 📂 Estrutura do Projeto
- **PluginsTcc** → Código dos plugins criados para o Dynamics 365.
- **Documentação** → Arquivos e instruções relacionadas ao TCC.
